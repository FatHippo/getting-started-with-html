Important Basic Concepts


The Web

The web is made up of mainly HTML documents. HTML stands for Hyper Link Markup Language. The web contains anything, like pictures, videos, music, or text. Browsers send requests using protocals, called HTTP, to servers which respond with files that your web browser displays.

HTML Intro

HTML is made up of:
Text Content-What you see
Markup-What it looks like
References to other documents-E.g. videos, images

Tags

What are tags?

HTML Markup is made up of tags. Tags are structure by starting with a less than symbol <, and then the tag name, finishing with a greater than symbol >. Then you put the contents. After the contents put another < followed by a /, then the tag name and ending with a >. A visual example would be: <b> This is what the text would look like </b> or <tagname> contents</tagname>.

Some useful tags are:

<b> or <strong> which bolds something
<em> or <i> which italicizes something
<img src="URL", alt="TEXT", width="NUMBER", height="NUMBER">

Attributes

What are attributes?

Attributes have the format of: <TAG ATTR="value">contents</TAG>

Some useful attributes are:

<a href="URL">What is a URL?</a> which sends you to the url when clicking on the words: "What is a URL?"
